# Hex Map Maker

A WebGL application for creating and editing hexagonal maps using Three.js.

## Features

- Interactive hex grid
- Click on hexes to change their color randomly
- 3D visualization with WebGL

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm start
   ```

3. Open your browser and navigate to `http://localhost:3000`

## Usage

- Click on any hexagon to change its color.
- Use mouse to rotate/zoom the view (if controls are added).

## Technologies

- Three.js for WebGL rendering
- Vanilla JavaScript
- HTTP Server for local development
